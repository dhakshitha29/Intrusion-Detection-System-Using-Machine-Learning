# Intrusion Detection System (IDS)

A host-based Intrusion Detection and Prevention System built using Python.  
The system monitors file system activity, running processes, and network connections, and detects anomalies using machine learning.

## Features
- File system monitoring using watchdog
- Process and network monitoring using psutil
- Anomaly detection using Isolation Forest
- Real-time alerts and logging

## Technologies Used
- Python
- psutil
- watchdog
- scikit-learn

## How to Run
```bash
pip install -r requirements.txt
python idps.py
